🚀 BizTrack Setup Guide

Google Drive Sync activate karo — sirf 10 minutes mein

⚠️ Pehle ye padho Ye steps sirf ek baar karne hain. Baad mein automatically sync hoga. Koi payment nahi, Google ka free plan use ho raha hai.
Step 1

Google Cloud Console open karo

Apne browser mein ye link open karo (same Google account se login karo jo app mein use karna hai):

https://console.cloud.google.com/
Tip: Agar pehli baar hai — "Terms of Service" accept karo, free hai.
Step 2

Naya Project banao

  1. Top-left mein "Select a project" dropdown click karo
  2. "New Project" click karo
  3. Project name type karo: BizTrack
  4. "Create" click karo
  5. Project create hone ke baad uspe click karo (select karo)
Step 3

Google Drive API Enable karo

  1. Left side menu mein "APIs & Services" → "Library" click karo
  2. Search box mein type karo: Google Drive API
  3. Result mein "Google Drive API" click karo
  4. "Enable" button click karo
✓ Done sign: Enable ke baad "Manage" button dikhega.
Step 4

OAuth Consent Screen setup karo

  1. Left menu: "APIs & Services" → "OAuth consent screen"
  2. User Type: "External" select karo → "Create"
  3. App name: BizTrack
  4. User support email: apna email select karo
  5. Developer contact email: apna email type karo
  6. "Save and Continue" click karo (baaki sab default rehne do)
  7. Scopes page pe bhi "Save and Continue"
  8. Test Users pe apna email add karo → "Save and Continue"
Step 5

OAuth Client ID banao ← Sabse important step

  1. Left menu: "APIs & Services" → "Credentials"
  2. Top pe "+ Create Credentials" → "OAuth client ID"
  3. Application type: "Web application"
  4. Name: BizTrack Web
  5. "Authorized JavaScript origins" mein Add karo:
http://localhost
http://127.0.0.1

Agar ye app kisi website pe host karo toh woh URL bhi add karo (e.g. https://yoursite.com)

  1. "Create" click karo
  2. Ek popup aayega — usme "Your Client ID" copy karo
Client ID aisa dikhta hai:
123456789-abcdefghijklmnop.apps.googleusercontent.com
Step 6

Client ID BizTrack mein paste karo

gdrive.js file open karo (Notepad se). Pehli line dikhegi:

const GDRIVE_CLIENT_ID = 'PASTE_YOUR_CLIENT_ID_HERE';

PASTE_YOUR_CLIENT_ID_HERE hata ke apna Client ID paste karo:

const GDRIVE_CLIENT_ID = '123456789-abcdef.apps.googleusercontent.com';

File save karo (Ctrl+S).

Step 7

App kaise use karein — Offline Install

💻 Laptop/PC pe (Chrome/Edge):

  1. BizTrack folder mein index.html double-click karo (Chrome mein open hoga)
  2. Address bar mein right side pe Install icon (⊕) dikhega
  3. Click karo → "Install" — Desktop pe icon ban jaayega
  4. Ab directly desktop se open kar sakte ho, browser ki zaroorat nahi

📱 Mobile pe (Android Chrome):

  1. App kisi free hosting pe upload karo (Netlify Drop — agle step mein bataya)
  2. Chrome mein website open karo
  3. Three dots (⋮) → "Add to Home screen"
  4. Home screen pe icon ban jaayega — bilkul app jaisa
Free Hosting (Netlify Drop): netlify.com/drop pe jaao → BizTrack folder drag & drop karo → free URL milega → mobile pe us URL se open karo → install karo.
Step 8

Drive Sync kaise use karein

Har page ke neeche ek blue sync bar dikhega:

  • "☁ Save to Drive" — Data Google Drive mein save karo (ek click)
  • "↓ Restore from Drive" — Dusre device pe ye click karo data aane ke liye
Best Practice: Koi bhi important kaam karne ke baad "Save to Drive" click karo. Dusre device pe kholo toh automatic pull ho jaata hai (pehle se logged in ho toh).

🎉 Setup Complete!

Ab tumhara BizTrack lifetime free hai, kisi bhi device pe chalega, aur Google Drive mein safe rahega.

Data loss ka koi chance nahi — Drive mein backup hamesha rahega.
